Для сборки используется:

ruby 1.8.6 (2010-02-04 patchlevel 398) [i386-mingw32]
buildr (1.4.4 x86-mswin32)

После установки Ruby и Buildr запустите buildr clean package, после скачивания нобходимых модулей зависимостей и сборки .war файл будет находиться в папке target.
Или можно использовать уже собраный файл, лежащий там же.

Для запуска используется Apache Tomcat 6.0. Просто скопируйте RSSReader-1.3.war в папку webapps установки Tomcat.

База данных MySQL используется с дефолтными настройками, по адресу jdbc:mysql://localhost:3306/rss. Нужно создать юзера rssreader с таким же паролем rssreader. После этого можно создать схему rss и нужные таблицы, а также добавить тестовые данные. Для этого можно использовать скрипты из RSSReader_preview\Persistence\db-scripts\

Если Tomcat запущен с дефолтными настройками порта, то вебсервис доступен по адресу http://localhost:8080/RSSReader-1.3/
